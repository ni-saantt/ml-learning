Student Performance Prediction using Multiple Linear Regression

This project predicts students' final exam scores using a Multiple Linear Regression model.
The prediction is based on study habits and academic 

Dataset
The dataset contains the following features:
- Study hours per day
- Attendance percentage
- Previous exam score
- Sleep hours
- Extra classes attended

Target variable:
- Final exam score

Model
- Multiple Linear Regression
- Train-Test split (80% training, 20% testing)

Evaluation
- Model performance is evaluated using RÂ² score.
